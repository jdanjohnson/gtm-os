# Channel rules — SEO

## Workflow

Always start GSC-first:

1. Pull `GSC_SEARCH_ANALYTICS` for the last 28 days.
2. Diagnose: low CTR with good position (rewrite metadata) vs. good CTR with bad
   position (improve content / add internal links).
3. Validate volume + intent in Semrush or your keyword tool of choice.
4. Decide: improve an existing page (preferred) vs. create a new page.
5. Use PostHog (or your analytics) after the click — does the page promise match the
   product?

## Hard don'ts

- Don't create a new page when an existing one already has impressions for the keyword.
- Don't target keywords with no ICP-fit search intent ("free", "tutorial", "vs.
  competitor" can be traps).
- Don't optimize for volume when you don't have conversion data on the page yet.
- Don't rewrite a page that's currently bringing pipeline without a clear hypothesis.
