# Landing Page Conversion Experiment

Run conversion copy experiments on landing pages — test headlines, CTAs, social proof, page structure — using frameworks like PAS, AIDA, StoryBrand.

## Outcome

**Goal:** Increase landing page conversion rate through systematic copy and structure experiments

## Hypothesis

Landing pages written with PAS framework and specific social proof convert at 2x the rate of generic feature-focused pages.

## Success Criteria

- **conversion_rate**: >5% visitor-to-signup
- **bounce_rate**: <40%
- **time_on_page**: >90 seconds average

## Knowledge Required

- **copywriting-frameworks**: PAS, AIDA, StoryBrand, Before-After-Bridge frameworks with examples *(required)*
- **brand-voice**: Tone, vocabulary, and messaging guidelines *(required)*
- **conversion-benchmarks**: Industry benchmarks for landing page metrics

## Workflows

- `content-write-optimize-publish`

## Skills

- `landing-page-copywriter`
- `seo-optimizer`
- `brand-voice-analyzer`

## Tools

- **web-search**: Research competitor landing pages for positioning
- **llm-scraper**: Extract competitor copy patterns and social proof

## Agent Config

- **Role:** Conversion Copywriter
- **Process:** sequential
- **Phases:** design, build, execute, measure, learn

## Experiments

### Framework Test

PAS vs AIDA vs StoryBrand for same product, measure conversion rate

### Headline Test

Benefit-led vs pain-led vs question-led headlines, measure click-through

### Social Proof Test

Customer logos vs testimonials vs case study stats, measure trust and conversion

### CTA Test

Start Free Trial vs See Demo vs Get Started — measure click rate
